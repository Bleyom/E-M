### Installation


##### Clone the repo
```
git clone https://github.com/Bleyom/dotfiles
```

##### Go to the "dotfiles" directory

```
cd dotfiles
```

##### Give execution permissions to necessary files

```
chmod +x scripts/*
chmod +x eww/scripts/*
```
